# TimeTravel Agency

Une webapp moderne et interactive pour une agence de voyage temporel de luxe, avec chatbot IA intégré et quiz personnalisé.

## ✨ Fonctionnalités

- **Design Premium** : Interface élégante avec thème sombre, accents dorés et animations fluides.
- **Quiz de Recommandation** : Un quiz interactif pour découvrir votre destination temporelle idéale.
- **3 Destinations Temporelles** :
  - **Paris 1889** : La Belle Époque et l'Exposition Universelle.
  - **Crétacé -65M** : L'aventure ultime au temps des dinosaures.
  - **Florence 1504** : L'âge d'or de la Renaissance italienne.
- **Chatbot IA** : Assistant virtuel intelligent (Mistral AI) pour répondre à toutes vos questions.
- **Animations Modernes** : Utilisation de **Framer Motion** pour une expérience utilisateur dynamique.
- **Formulaire de Réservation** : Système de réservation avec calcul de prix en temps réel.
- **Responsive Design** : Optimisé pour mobile, tablette et desktop.

## 🛠️ Technologies

- **Frontend** : React, TypeScript, Tailwind CSS
- **Animations** : Framer Motion
- **IA/Backend** : Supabase (Edge Functions), Mistral AI
- **Outils** : Vite, Lucide React

## 🚀 Démarrage rapide

1. **Installation** :
```bash
npm install
```

2. **Configuration** :
Copiez `.env.example` vers `.env` et remplissez vos clés Supabase.
```bash
cp .env.example .env
```

3. **Développement** :
```bash
npm run dev
```

## 🤖 Intelligence Artificielle

Le chatbot est alimenté par **Mistral AI** via une Edge Function Supabase. Il est configuré avec une personnalité d'expert historien pour conseiller les clients sur les destinations, les prix et la sécurité des voyages temporels.
